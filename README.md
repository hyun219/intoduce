# Introduce
## 저는 인간의 내면을 문학적으로 그려내는 이소현입니다. 
 저에게 문학이란 다양한 인간의 삶과 내면을 볼 수 있는 통로입니다. 문학에 이어 연극/영화에 관심을 갖게 된 것도 이와 같은 이유였습니다. 문학은 인간을 언어로 그려낸다면, 연극과 영화는 살아있는 몸짓, 동작, 생생한 온기로 직접 전달합니다. 그러한 점에서 두 영역은 같은 곳을 향한다고 느꼈습니다. 
 
 연극 동아리의 음향 스태프로 참여한 것에 이어 배우로서 직접 무대에 서게 되었고, 이 과정에서 저는 갚진 시행착오를 겪었습니다. 초반에는 인물의 감정을 텍스트 그대로만 분석해서 연기와 인물의 서사가 연결되지 않은 채 겉도는 한계를 마주했습니다. 하지만 탄탄한 기초를 위해 다시 처음으로 돌아갔습니다. 나의 입장이 아닌 타인의 삶을 온전히 이해하고 느끼기 위해 인물의 전사와 심리를 문학적으로 세세하게 분석했습니다. 이 덕분에 서사에 대한 입체적인 분석 능력을 기를 수 있었으며 동료들과의 합을 맞추는 데에도 도움이 되었습니다. 
 
 이처럼 타인의 복합적인 감정을 이해하고 나의 것으로 체계화하기 위해 노력했던 시간은 저의 창작 세계를 풍성하게 했으며, 이는 학과 백일장에서 인간의 섬세한 내면 표현을 담아낸 작품의 수상을 통해 성장의 결과물을 내보였습니다. 이후에도 독립영화의 조연출로서 영화 제작 과정에 직접 참여하며 인간을 담아낼 수 있는 여러 영역에 대한 관심은 더욱 확장되었습니다. 
 
 앞으로도 문학, 연극, 영화의 각 특성을 통해 인간의 내면을 깊이 있게 담아내어 인간의 삶을 다양한 방식으로 표현하는 창작자가 되고자 합니다.
